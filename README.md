# ReliaVue

ReliaVue is a native AWS-hosted geospatial AI platform for rooftop and solar PV anomaly detection, localization, scoring, and propagation forecasting.

## Build Doctrine
- Backend-first, schema-first, infrastructure-first
- No shell pages
- One module at a time
- Every milestone must pass a verification gate before expansion
- Every verified milestone gets a protected checkpoint tag

## Environments
- local
- dev
- stage
- prod

## Initial Milestones
- Milestone 0: Governance and source of truth
- Milestone 1: AWS foundation stack
