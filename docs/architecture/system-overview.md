# System Overview

## Purpose
ReliaVue is a native geospatial AI system for rooftop and solar PV anomaly detection, localization, condition scoring, and propagation forecasting.

## Non-Negotiables
- no shell-only UI
- no disconnected backend
- one source of truth
- milestone gates required

## Core Services
- api service
- worker service
- spatial localization engine
- anomaly ontology engine
- scoring engine
- reporting engine
